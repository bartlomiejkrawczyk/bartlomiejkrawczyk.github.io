## Profile Provisioning

It allows for 2 methods of profile provisioning:

1. Manual - you need to manually copy Secret from mobile app to web application to map profile to domain account.
2. Using QR code - after scanning the QR code from web application profile will be connected with specific domain account that is automatically created.
