# Dynamic terminal

Terminal pops up on `ESC` or `ctrl` + `shift` + `p` (vsc like)

Terminal has support for command filtering.

Commands range:

- print support
- link to my other socials
- modes - like tag filtering / project filtering
- in the future I will add a lot more :D

# Two Language support

- English (main language)
- Polish (native - but here secondary)

# Markdown support

Astro.js allows for nice markdown support

all the github flavored markdown spec is met here
